# 🔍 Railway - Где найти кнопку добавления базы данных

## Ситуация: У вас создан проект, но не видите где добавить PostgreSQL

---

## ✅ РЕШЕНИЕ 1: Через Canvas (основной способ)

1. **Убедитесь что вы внутри вашего проекта**
   - Вверху должно быть название: `french-course-platform` (или похожее)

2. **Посмотрите на экран - там должен быть "холст" (Canvas)**
   - Вы увидите прямоугольник с вашим сервисом
   - Это визуальное представление вашего проекта

3. **Найдите кнопку добавления:**

   **Вариант А:** Кнопка в правом верхнем углу Canvas:
   ```
   [+ New]  или  [+ Add]  или  [New +]
   ```

   **Вариант Б:** Правый клик на пустом месте Canvas:
   - Кликните правой кнопкой мыши на пустое место
   - Появится меню с опцией "Add Service" или "New Database"

   **Вариант В:** Кнопка снизу Canvas:
   ```
   [+ Add Service]
   ```

4. **В появившемся меню выберите:**
   ```
   Database → PostgreSQL
   ```

---

## ✅ РЕШЕНИЕ 2: Через боковое меню

1. **Посмотрите слева - там должно быть меню проекта**

2. **Найдите пункт "Services" или список сервисов**
   - Там будет ваш основной сервис (french-course-platform)

3. **Найдите кнопку добавления:**
   - Может быть значок "+" рядом с Services
   - Или кнопка "Add Service"

4. **Выберите: Database → PostgreSQL**

---

## ✅ РЕШЕНИЕ 3: Через верхнее меню

1. **Посмотрите в правый верхний угол всей страницы**

2. **Найдите кнопку:**
   ```
   [New +]  или  [+ New]  или  [Create New]
   ```

3. **В выпадающем меню выберите:**
   ```
   New Database → PostgreSQL
   ```

---

## ✅ РЕШЕНИЕ 4: Альтернативный путь (если ничего не работает)

Если вы вообще не видите опции добавления базы:

1. **Проверьте что вы смотрите на правильный проект:**
   - URL должен быть: `railway.app/project/[ваш-id]`

2. **Попробуйте обновить страницу (F5)**

3. **Убедитесь что проект создан и деплой завершился:**
   - Посмотрите на статус деплоя
   - Должен быть зелёный индикатор

4. **Если всё равно не видите - создайте через Dashboard:**
   - Вернитесь на главную страницу Railway
   - Нажмите "New Project"
   - Выберите "Provision PostgreSQL"
   - Затем подключите его к вашему сервису

## ⚡ БЫСТРЫЙ СПОСОБ (через URL):

Попробуйте открыть напрямую:
1. Скопируйте ID вашего проекта из URL
   - Пример: `railway.app/project/abc123def456`
   - ID = `abc123def456`

2. Откройте:
   ```
   https://railway.app/new/database/postgres?projectId=ваш_id
   ```

Это должно открыть форму создания PostgreSQL сразу!
